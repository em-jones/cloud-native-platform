# Frequently asked questions

## Why Kubernetes

Kubernetes earned the reputation as the most well-known, well-supported,
and cloud-agnostic container orchestration platform.
It has a large and active community, extensive documentation, and a rich
ecosystem of tools and extensions.
Kubernetes provides robust features for:

- scaling
- self-healing
- rolling updates
- service discovery
- load balancing
- secret management
- configuration management
- storage orchestration
- network-access control

Additionally, Kubernetes has become the most widely adopted container orchestration
platform for open-source projects and enterprises, making it a safe choice for
long-term projects.
